### Installation

Installation is handle via [npm](https://docs.npmjs.com/)

To create a new project based from this repo, simply run:

```shell
npm i
```

---

### Running

```shell
npm run start

Runs the app in the development mode.
Open http://localhost:3000 to view it in your browser.
```
